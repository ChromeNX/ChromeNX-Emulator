<h1>ChromeNX Emulator | The First of its Kind </h1>
<p>ChromeNX is the first of its kind. A emulator built just for Chromebooks. It will be able to run and optimize Nintendo Switch games at a high rate. Built off of the already powerful Yuzu Emulator, it will boast huge power and redefine the emulating industry.</p>

* Seamless integration with Chromebook

* Wide range of supported games

* User-friendly interface

<h1>Why I started this Project</h1>

<p>I decided to start this project when I was bored in school and realized there were no Switch Emulators for Chromebooks. 
I then decided to set off and create the first one there was. I am currently still working on this project.</p>
